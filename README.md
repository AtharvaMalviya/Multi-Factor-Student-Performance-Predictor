# Multi-Factor Student Performance Predictor 🎓

## Problem
Student performance depends on multiple factors like study time, attendance, and sleep. However, students often fail to understand how these factors affect their scores.

## Solution
This project uses Multiple Linear Regression to predict student performance based on three inputs:
- Study hours
- Attendance percentage
- Sleep hours

## Tech Stack
- Python
- NumPy
- Scikit-learn
- Matplotlib

## How to Run

1. Install Python on your system.

2. Make sure the required libraries are installed:
   pip install numpy scikit-learn matplotlib

3. Copy the code from `main.py`.

4. Paste and run it in any Python environment such as:
   - Jupyter Notebook  
   - VS Code  
   - Google Colab  

5. Enter the input values when prompted.

6. View the predicted score and graph output.

## Output
- Predicted exam score
- Graph showing relationship between study hours and score

## Example
Input:
Study Hours = 5  
Attendance = 75  
Sleep = 7  

Output:
Predicted Score ≈ 60
